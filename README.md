# redict-qt5

![](https://api.travis-ci.org/rekols/redict-qt5.svg?branch=master)

A chinese dictionary based on C++/Qt development.

## Build dependencies

`sudo apt install g++ qt5-default`

## Installation

* mkdir build
* cd build
* qmake ..
* make

## Usage

* ./redict-qt5
